# Отчет по лабораторной работе 5

## Тема
Анализ текстового файла: подсчет строк, слов и символов с обработкой ошибок ввода.

## Постановка задачи
1. Запросить у пользователя имя (или путь) текстового файла.
2. Открыть файл в режиме чтения и собрать его содержимое.
3. Подсчитать:
   - количество строк;
   - количество слов (слова разделяются пробелами);
   - количество символов (включая пробелы и символы новой строки).
4. Вывести эти статистики в консоль.
5. Обработать исключение `FileNotFoundError`, если файл не существует.

## Структура решения
- Файл `main.py` содержит функцию `main()`, которая инкапсулирует всю логику программы и вызывается при запуске скрипта.

## Подробности реализации

### Функция `main()`
```python
def main():
    filename = input("Введите имя файла: ").strip()

    try:
        with open(filename, "r", encoding="utf-8") as file:
            lines = file.readlines()
    except FileNotFoundError:
        print("Файл не найден.")
        return

    line_count = len(lines)
    word_count = sum(len(line.split()) for line in lines)
    char_count = sum(len(line) for line in lines)

    print(f"Количество строк: {line_count}")
    print(f"Количество слов: {word_count}")
    print(f"Количество символов: {char_count}")
```

- **Ввод**: пользователю предлагается ввести название файла. Метод `strip()` удаляет лишние пробелы в начале и конце строки.
- **Чтение файла**: используется менеджер контекста `with open(..., encoding="utf-8")`, что гарантирует закрытие файла после чтения. Весь файл загружается в список `lines`.
- **Обработка ошибок**: блок `try / except FileNotFoundError` выводит сообщение «Файл не найден.» и завершает работу функции, если путь указан неправильно.
- **Подсчеты**:
  - `len(lines)` — количество строк;
  - `sum(len(line.split()) for line in lines)` — количество слов (разделитель — пробел);
  - `sum(len(line) for line in lines)` — количество символов, включая пробелы и переходы на новую строку.
- **Вывод**: результаты печатаются в удобном для чтения формате.
